# The Odds API - C# / ASP.NET Sample

A simple web application demonstrating how to use [The Odds API](https://the-odds-api.com) with C# and ASP.NET Core.

## Usage

1. Get your free API key at [the-odds-api.com](https://the-odds-api.com)
2. Replace `YOUR_API_KEY_HERE` in `Program.cs` (line 6) with your API key
3. Click "Run"
4. View the results in the browser

## What it does

- Fetches available sports from The Odds API
- Gets odds for upcoming matches
- Displays bookmaker prices in a simple web page

## Requirements

- .NET 8.0 SDK (included in Replit)

## Running locally

```bash
dotnet run --urls http://localhost:3000
```

Then navigate to http://localhost:3000
